Resource Links

Awk is a powerful and versatile text processing tool commonly used in Unix and Linux environments. It takes its name from the initials of its creators: Alfred Aho, Peter Weinberger, and Brian Kernighan. Awk is primarily used for handling and manipulating structured text data, such as log files, CSV files, and other tabular data.



Grep stands for global search for regular expression and print out. It is a function of PHP which prints lines matching a pattern and returns different variations based on command used.



Vim is a highly configurable text editor built to make creating and changing any kind of text very efficient. It is included as "vi" with most UNIX systems and with Apple OS X.



Bash short for "Bourne Again SHell," is a popular Unix shell and command-line interpreter used in Unix, Linux, and macOS systems. It is the successor to the original Bourne shell (sh) and is the default shell for most Linux distributions.



Virtual Machines (VMs) are software-based emulations of physical computers that run an operating system (OS) and applications just like a physical machine. VMs allow multiple virtualized operating systems to coexist and run on a single physical host machine simultaneously. Each VM operates independently and has its own virtual hardware resources, such as CPU, memory, disk space, and network interfaces.



Basic Commands The command-line interface (CLI) is a text-based interface that allows users to interact with a computer's operating system or applications by typing commands in a terminal or command prompt. The command-line interface provides a way to control the computer and execute various tasks using text commands rather than a graphical user interface (GUI).
