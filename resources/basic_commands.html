The Command-Line Interface (CLI)


The CLI, often referred to as a terminal, shell, or console allows you to interact with your computer through a text based interface. This is very different from the graphical user interface (GUI) that you might be used to when using your own PC. The CLI provides the opportunity to complete certain tasks much faster than you would be able to using a traditional GUI. The table below shows a list of some of the basic commands you might find useful as you adjust to using the CLI.

CLI Commands

Information was pulled from hostinger.com.
The following commands are case-sensative.

CommandShort for...EffectExamples
sudoSuperuser doAllows you to perform tasks that require administer or root permissions.
  • sudo rmdir -p mygrades/bad-grades
  • sudo rmdir -p game-of-thrones/season-8
pwdPrint Working DirectoryDisplays the path to your current working directory
  • pwd
  • pwd -L (Prints enviroment variable content, including links)
  • pwd -P (Prints the actual path of the current directory)
cdChange DirectoryAllows you to navigate the file structure. You can put in a full path or a specific directory in your current working directory.
  • cd documents/plans/secrect-plans/
  • cd life-goals/make-it-rain
lsListShows the content of the current working directory. It can also be followed by a path to list the contents of a specific directory.
  • ls
  • ls area-51/aliens
catConcatenateLists, combines, and writes file content to the standard output
  • cat santas-naughty-list.txt
  • cat Bill-Gates-Fortune.txt My-Fortune.txt > genies-todo-list.txt
cpCopyCopies files or directories and their content.
  • cp document
  • cp folder
mvMoveMoves or renames a file or directory.
  • mv get-rich-quick.txt home/trash
mkdirMake DirectoryCreates a new directory.
  • mkdir labs/lab10
  • mkdir cs2210/hw
rmdirRemove DirectoryDeletes a directory.
  • sudo rmdir my-jokes
rmRemoveRemoves files in a directory.
  • rm my-wallet/money
  • rm broccoli
grepGlobal Regular Expression PrintSearches the text in a given directory for a key word.
  • grep meaning life.txt
  • grep money wallet.txt
