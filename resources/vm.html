What is a Virtual Machine?

Virtual Machine

  • It "is a digital version of a physica computer"
  • It functions much like a physical computer running
    programs, running its own operating system,
    having its own storage, running APPs.
  • Virtual machines run on a physical maichine. If
    fact, one single computer which is often a server can
    serve as the host of multiple virtual machines.
  • Hypervisor is the software that is used on a
    single, physical computer to run multiple virtual machines
