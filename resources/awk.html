from the AWK user guide: The basic function of awk is to search files for lines (or other units of text) that contain certain patterns. When a line matches one of the patterns, awk performs specified actions on that line. awk continues to process input lines in this way until it reaches the end of the input files.

awk -F ',' 'BEGIN {sum = 0; count = 0} {sum += $2; count++} END {avg = sum / count; print "Average: " avg}' data.csv
